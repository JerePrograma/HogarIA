# Control mensual de caja

## Definiciones

- **Salida bruta de caja**: toda salida real del mes (consumo, deuda, ahorro, inversión, desembolsos recuperables, transferencias externas).
- **Gasto económico real**: consumo + deuda financiera - reintegros (sin tratar ahorro/inversión/capital prestado como pérdida definitiva).
- **Recupero de capital**: retorno de principal prestado; mejora caja pero **no es ingreso económico**.
- **Interés cobrado**: sí es ingreso/ganancia.

## Ejemplo

- Ingreso salarial: +1000
- Comida: -200
- Ahorro: -100
- Préstamo otorgado (DISBURSEMENT): -150
- Recupero principal: +80
- Interés cobrado: +20

Resultados:
- Salida bruta = 450
- Gasto económico = 200
- Ingreso económico = 1020 (1000 + 20)
- Neto de caja = 1000 + 80 + 20 - 450 = 650

## Por qué no netear todo automáticamente

Netear todo oculta disciplina financiera y riesgo de liquidez. Por eso se separa:
- caja (flujo bruto y neto),
- resultado económico,
- recuperos de capital.
